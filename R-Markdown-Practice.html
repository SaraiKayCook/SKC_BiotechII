1. Summary

I used data on 50,000 diamonds. I chose not to print the individual data in this markdown report because it would be way too long.

2. Stacked Bar Chart

This chart shows how many diamonds are of each cut and color.

3. Pie Chart

This chart shows the proportion of each diamond cut. I have included the source code so you can see how I made it.

ggplot(diamonds, aes(x = "", fill = cut)) +
  geom_bar(width = 1) +
  coord_polar(theta = "y") +
  theme_void() +
  labs(title = "Proportion of Diamonds Per Cut",
       fill = "Cut")

4. Another Way of Visualizing Data

You can also enter this code into R Studio to see another way of visualizing how many diamonds there are per cut.

ggplot(diamonds, aes(x = cut, fill = cut)) +
  geom_bar() +
  labs(x = "Cut",
       y = "Number",
       title = "Number of Diamonds per Cut",
       fill = "")
